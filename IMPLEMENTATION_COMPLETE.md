## How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/ishaniff345-design/email-segregation-mvp.git
cd email-segregation-mvp
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Start the server
```bash
python main.py
```

### 4. Open in browser
```
http://127.0.0.1:8000
```
